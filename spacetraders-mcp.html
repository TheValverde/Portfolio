SpaceTradersMCP

A comprehensive Model Context Protocol (MCP) server for interacting with the SpaceTraders API. This project exposes SpaceTraders API endpoints as MCP tools, allowing AI agents to manage agents, fleets, contracts, and trading operations in the SpaceTraders universe. The implementation demonstrates advanced integration of AI agents with gaming APIs, enabling intelligent automation and decision-making within complex game environments.

Project Overview

This MCP server enables AI agents to:

  • Register and manage SpaceTraders agents
  • View and manage ships and fleets
  • Accept, deliver, and fulfill contracts
  • Trade resources and interact with markets
  • Explore systems, waypoints, and shipyards

The implementation is based on the MCP standard, using the mcp Python package for server infrastructure.

Key Features Implemented

Agent Management

  • Register new SpaceTraders agents
  • View agent details and list agents

Contract Operations

  • List, view, accept, deliver, and fulfill contracts

Ship & Fleet Management

  • List ships, view ship details, purchase ships
  • Navigate, dock, orbit, refuel, and extract resources with ships
  • Transfer and sell cargo

Exploration & Trading

  • Scan systems, waypoints, and ships
  • View markets and shipyards
  • Chart waypoints and refine resources

Technical Implementation

Architecture

  • MCP server built with Python 3.12+
  • Docker containerization for easy deployment
  • Environment-based configuration management
  • Persistent agent token management

Project Structure

  • src/main.py — Main MCP server implementation and tool definitions
  • src/spacetraders_utils.py — SpaceTraders API client and token management
  • src/utils.py — Utility functions
  • agent_tokens.json — Persistent agent tokens (auto-managed)
  • schemas/ — OpenAPI and reference JSON files for SpaceTraders API

Technologies Used

  • Python 3.12+
  • MCP (Model Context Protocol)
  • Docker
  • uv (Python package manager)
  • SpaceTraders API v2
  • OpenAPI Specification
  • JSON Schema
  • Environment Configuration

Installation & Usage

Prerequisites

  • Python 3.12+
  • SpaceTraders API key (from my.spacetraders.io)
  • Docker (optional, for containerized deployment)

Quick Start

  1. Install uv: pip install uv
  2. Clone repository: git clone https://github.com/TheValverde/spacetraders-mcp.git
  3. Install dependencies: uv pip install -e .
  4. Configure environment: SPACETRADERS_API_KEY=your-api-key-here
  5. Run server: uv run src/main.py
