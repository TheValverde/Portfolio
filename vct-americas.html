VCT Americas 2023

As one of the lead programmer for the VCT Americas real-time graphics package, I had the opportunity to work with Riot Games, creating dynamic and immersive visuals for the live show. This project showcases my expertise in real-time graphics programming, utilizing cutting-edge software like Ventuz 6.12 Designer.

Responsibilities

  • Programming dynamic graphic templates for live operators to use during live shows
  • Ensuring templates were configured for user-friendly interaction with Ventuz Director
  • Collaborating with the production team to ensure smooth integration of graphics during broadcasts
  • Debugging and optimizing graphics performance for real-time rendering

Technologies Used

  • Ventuz 6.12 Designer
  • Ventuz Director
  • Real-time data integration
