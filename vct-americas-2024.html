VCT Americas 2024

Collaborated with Riot Games as a Lead Programmer for the VCT Americas data-driven real-time graphics package. My primary responsibility was to develop dynamic, data-driven graphic templates that live operators could efficiently utilize during live broadcasts. I employed Ventuz 7 Designer to create the templates and ensured they were configured for seamless and user-friendly interaction with real-time data updates within the Ventuz Director environment. This project included a 3D previz for Operators to get a better grasp of the final look on screen within a 3D environment that replicates the real environment where the graphics would be.

Responsibilities

  • Leading programming for VCT Americas data-driven real-time graphics package
  • Developing dynamic, data-driven graphic templates for live operators
  • Creating 3D previz environment for operator training and visualization
  • Ensuring seamless integration with Ventuz Director environment
  • Optimizing templates for real-time data updates during live broadcasts

Technologies Used

  • Ventuz
  • Ventuz Designer
  • Systems Design
  • Remote Teamwork
  • 3D Visualization
