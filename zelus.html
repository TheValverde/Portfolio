Zelus

First release Oct 2023, Zelus is a projection system that takes data from the match, and projects it onto the ice in an end-user friendly fashion during Period Breaks for NHL games. included features I programmed such as the Shot Report and Leaderboard. Second update, April 2024: This new update featured Rewind, another feature I programmed, which allows for a play, chosen by the operator, to be replayed on the ice Third update, October 2024: The Playlist Update! Created new templates, wrote user manual, and implement a new playlist approach into existing programming.

Responsibilities

  • Developing real-time projection system for NHL games during period breaks
  • Creating Shot Report and Leaderboard features for live game data display
  • Implementing Rewind feature for replaying selected plays on ice
  • Designing and programming new templates for enhanced visual experience
  • Writing comprehensive user manual for operators
  • Implementing playlist approach for streamlined content management
