Zelus

Zelus is a powerful real-time stats visualizer and projection system developed for NHL games. This innovative system takes live match data and projects it directly onto the ice surface during period breaks, creating an immersive fan experience. As a Real Time Developer at Quince Imaging, I played a key role in programming core features and managing multiple system updates over the project's lifecycle.

Project Timeline & Updates

First Release - October 2023

Initial launch of Zelus projection system with core functionality for NHL period break displays.

Second Update - April 2024

Major feature expansion including the Rewind functionality for replaying selected plays on ice.

Third Update - October 2024

The Playlist Update featuring new templates, comprehensive user manual, and streamlined playlist approach.

Key Features I Programmed

  • Shot Report: Real-time display of shooting statistics and analytics
  • Leaderboard: Live ranking system showing top performers during the game
  • Rewind Feature: Interactive replay system allowing operators to replay selected plays on ice
  • Playlist System: Streamlined content management for organized display sequences
  • Template Engine: Customizable visual templates for different content types
  • User Interface: Operator-friendly controls for live event management

Technical Responsibilities

Project Impact

Zelus represents a significant advancement in live sports entertainment technology, bringing real-time data visualization directly to the ice surface. The system enhances fan engagement during NHL games by providing interactive statistics and replay capabilities that were previously impossible to display in such an immersive format.

The multi-year development cycle demonstrates the project's success and continued evolution, with each update adding substantial new functionality while maintaining system stability in high-pressure live event environments.
