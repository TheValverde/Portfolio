LCS Spring Split 2023

As a lead programmer for the LCS Spring Split 2023 real-time graphics package, I collaborated with Riot Games to develop captivating and interactive visuals for the League of Legends Championship Series. Utilizing state-of-the-art tools like Ventuz 6.12 Designer, I crafted user-friendly, data-driven templates that integrated seamlessly with the Ventuz Director.

Responsibilities

  • Creating dynamic graphic templates for live operators to use during live shows
  • Ensuring templates were configured for user-friendly interaction with Ventuz Director
  • Collaborating with the production team to ensure smooth integration of graphics during broadcasts
  • Debugging and optimizing graphics performance for real-time rendering

Technologies Used

  • Ventuz 6.12 Designer
  • Ventuz Director
  • Real-time data integration
